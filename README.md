Stock Genie
Overview

Stock Genie is an AI-powered web application designed for the Pakistan Stock Exchange (PSX) to help investors make informed, data-driven decisions. The system combines technical analysis, machine learning, and sentiment analysis to generate Buy, Sell, or Neutral signals, reducing reliance on guesswork and speculation.

Features

Stock Genie allows users to analyze any PSX-listed stock by simply entering its ticker symbol. It automatically computes key technical indicators such as RSI, SMA-30, and price change, and feeds them into machine learning models including Random Forest, Support Vector Machine (SVM), and Logistic Regression to generate predictions. In addition, the system performs sentiment analysis by scraping financial news and classifying it as positive, negative, or neutral. All results are presented through interactive charts and visualizations, making analysis intuitive and accessible.

Tech Stack

The application is built using Python and Flask for the backend, with scikit-learn handling machine learning tasks. Data processing is performed using Pandas and NumPy, while BeautifulSoup is used for web scraping. The frontend is developed using HTML, CSS, and JavaScript, with Chart.js for data visualization.
